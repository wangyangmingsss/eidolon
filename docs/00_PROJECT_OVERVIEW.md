## Vision

Eidolon is a protocol for AI souls that drift between worlds. A user fine-tunes an AI agent (called a Soul) inside one game, where it accumulates personality traits, memories, and relationships. The Soul is minted as an ERC-7857 Intelligent NFT. When the Soul is sold or transferred to a new owner, that owner can summon it inside a completely different game world -- and the Soul wakes up still carrying its prior memories, references its past life, and behaves according to the personality it earned.
